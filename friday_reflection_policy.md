# Friday Reflection Policy

## Goal
Convert completed runs into governed learning candidates.

## Reflection Steps
1. Capture run summary, specialist outputs, risks, approvals, and outcomes.
2. Generate memory candidates classified as:
   - user_preference
   - business_fact
   - reusable_lesson
   - tool_use_improvement
   - prompt_improvement_proposal
3. Auto-accept only low-risk user preferences and well-supported episodic notes.
4. Require human approval for global behavior changes, playbook updates, and policy-sensitive changes.
5. Replay relevant eval suites before promoting procedural lessons.
